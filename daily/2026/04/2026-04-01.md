- [455.分发饼干](https://leetcode.cn/problems/assign-cookies/description/)
  - 先将孩子胃口数组和饼干尺寸数组都从小到大排序。
  - 然后从胃口最大的孩子开始遍历，同时用一个指针指向当前最大的未分配饼干。
  - 如果当前最大饼干可以满足当前孩子，就将这块饼干分给他，并把满足的孩子数加一，同时饼干指针左移。
  - 如果当前最大饼干都无法满足这个孩子，那么这个孩子一定无法被满足，因为剩下的饼干只会更小。
  - 这样就能保证尽量把大饼干留给大胃口孩子，从而得到最大满足人数。

- [53.最大子数组和](https://leetcode.cn/problems/maximum-subarray/description/)
  - 只保留“对后面有贡献”的前缀和
  - 用 count 表示当前连续子数组和，用 result 表示遍历过程中出现过的最大和。
  - 每次把当前元素加入 count，然后用 count 更新 result。
  - 如果 count 小于等于 0，我就把它重置为 0，因为一个非正的前缀和对后续子数组没有贡献，只会拖累后面的结果。
  - 最后返回 result 即可
    ```cpp
    class Solution {
    public:
    int maxSubArray(vector<int>& nums) {
        // result 用来记录目前为止遇到的最大子数组和
        // 这里必须初始化为很小的值，不能写成 0
        // 因为数组可能全是负数
        int result = INT32_MIN;

        // count 表示当前连续子数组的和
        int count = 0;

        // 遍历数组中的每一个元素
        for (int i = 0; i < nums.size(); i++) {
            // 把当前元素加入当前连续和
            count += nums[i];

            // 如果当前连续和比历史最大值更大，就更新答案
            if (count > result) {
                result = count;
            }

            // 如果当前连续和已经小于等于 0，
            // 说明这一段对后面的元素没有帮助，直接丢弃
            if (count <= 0) {
                count = 0;
            }
        }

        // 返回最大子数组和
        return result;
    }
    };
    ```
