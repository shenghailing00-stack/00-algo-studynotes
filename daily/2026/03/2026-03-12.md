- [151.反转字符串里的字母](https://leetcode.cn/problems/reverse-words-in-a-string/description/)
  - 将整个字符串全反转，再对每个单词反转，最后移除多余空格
  - 重点在用快慢指针移除空格
    - 快指针：获取符合题目要求的字母
    - 慢指针：将快指针指向的符合要求的字母赋值给慢指针（更新）
    - 思路参考[27.移除元素](https://programmercarl.com/0027.%E7%A7%BB%E9%99%A4%E5%85%83%E7%B4%A0.html#%E7%AE%97%E6%B3%95%E5%85%AC%E5%BC%80%E8%AF%BE)
      ```python
      # 将字符串拆分为单词，即转换成列表类型
        words = s.split()

        # 反转单词
        left, right = 0, len(words) - 1
        while left < right:
            words[left], words[right] = words[right], words[left]
            left += 1
            right -= 1

        # 将列表转换成字符串
        return " ".join(words)
      ```
  - [右旋字符串](https://kamacoder.com/problempage.php?pid=1065)
    - 先整体倒序输出，再将字串倒叙
      ```python
      #获取输入的数字k和字符串
      k = int(input())
      s = input()

      #通过切片反转第一段和第二段字符串
      #注意：python中字符串是不可变的，所以也需要额外空间
      s = s[len(s)-k:] + s[:len(s)-k]
      print(s)
      k = int(input())
      s = input()

      print(s[-k:] + s[:-k])# 从“倒数第 $k$ 个位置”开始，一直截取到最后+从最开头开始，一直截取到“倒数第 $k$ 个位置”（不包含它自己）
      ```
      ```python
      class Solution:
      def rightRotate(self, s: str, k: int) -> str:
        # 因为 Python 字符串不可变，我们先把它变成列表（如果在 C++ 里就不需要这一步，直接是 O(1)）
        s_list = list(s)
        n = len(s_list)
        
        # 翻转小助手：只要告诉它起点和终点，它就能把这一段原地翻转
        def reverse_part(left, right):
            while left < right:
                s_list[left], s_list[right] = s_list[right], s_list[left]
                left += 1
                right -= 1

        # 核心魔法开始，只用 3 行！
        # 1. 整体大翻转 (0 到末尾)
        reverse_part(0, n - 1)
        
        # 2. 翻转前 k 个 (0 到 k-1)
        reverse_part(0, k - 1)
        
        # 3. 翻转剩下的 (k 到末尾)
        reverse_part(k, n - 1)
        
        # 最后把原地修改好的列表重新变成字符串交卷
        return ''.join(s_list)
      ```
