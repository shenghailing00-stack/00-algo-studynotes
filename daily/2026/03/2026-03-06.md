 - 54.螺旋矩阵https://leetcode.cn/problems/spiral-matrix/description/（反过来）
   - **四边界收缩**——终于清楚了终于清楚了！这个方法比之前gemini给的方便多了！
     - 空值处理： 当 matrix 为空时，直接返回空列表 [] 即可。
     - 初始化： 矩阵 左、右、上、下 四个边界 l , r , t , b ，用于打印的结果列表 res 。
     - 循环打印： “从左向右、从上向下、从右向左、从下向上” 四个方向循环打印。
     - 根据边界打印，即将元素按顺序添加至列表 res 尾部。**matrix[t][i]:第top行第i列**
     - 边界向内收缩 1 （代表已被打印）。
     - 判断边界是否相遇（是否打印完毕），若打印完毕则跳出。
     - 返回值： 返回 res 即可。

     ```python
     class Solution:
     def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        if not matrix: return []
        l, r, t, b, res = 0, len(matrix[0]) - 1, 0, len(matrix) - 1, []
        while True:
            for i in range(l, r + 1): res.append(matrix[t][i]) # left to right
            t += 1
            if t > b: break
            for i in range(t, b + 1): res.append(matrix[i][r]) # top to bottom
            r -= 1
            if l > r: break
            for i in range(r, l - 1, -1): res.append(matrix[b][i]) # right to left
            b -= 1
            if t > b: break
            for i in range(b, t - 1, -1): res.append(matrix[i][l]) # bottom to top
            l += 1
            if l > r: break
        return res
     ```
- 前缀和思路练习：44.开发商购买土地https://kamacoder.com/problempage.php?pid=1044
  - 如果我们提前知道这块蛋糕总共有多少个草莓（总价值 total_sum），那么不管怎么切，我们只需要知道其中一半的数量就够了
  - **二维变一维**：把每一行每一列的总价值提前算出来
  - 差值：假设总草莓数是 total，你切下来的上半部分是 A。那么下半部分自然就是 total - A。它们俩的差值公式是：abs(A - (total - A))，化简，就变成了 abs(2 * A - total) （**abs** 是 Python 里求绝对值的函数，保证差值永远是正数）
    ```python
    import sys

    def main():
    # 1. 霸道总裁式读取：把屏幕上所有数字一次性全吸进来，切成小块
        input_data = sys.stdin.read().split()
        if not input_data:
            return
    
    # 2. 拿到这块“大蛋糕”的长和宽
        n = int(input_data[0])
        m = int(input_data[1])
    
    # 准备好三个记账本
        row_sums = [0] * n  # 记录每一行总和的压缩包
        col_sums = [0] * m  # 记录每一列总和的压缩包
        total_sum = 0       # 记录整块土地的总价值
    
    # 3. 开始遍历土地，把所有的草莓都数明白（建压缩包）
        idx = 2  # 前面两个数字是 n 和 m，所以从索引 2 开始取土地的数据
        for i in range(n):
            for j in range(m):
                val = int(input_data[idx])
            
            # 关键一步：在遍历每个格子的同时，直接把账算好
                row_sums[i] += val  # 把价值加到对应的“行账本”里
                col_sums[j] += val  # 把价值加到对应的“列账本”里
                total_sum += val    # 把价值加到“总账本”里
            
                idx += 1  # 指针往后走，取下一个格子的数据
            
    # 4. 准备一把尺子，记录历史最小差值（初始值设为无穷大！）
        min_diff = float('inf')
    
    # 5. 模拟【横着切】（切行）
        current_sum = 0
    # 注意：我们只能在两行之间切，所以最多切到倒数第二行（n-1），不能切在最后一行下面
        for i in range(n - 1): 
            current_sum += row_sums[i]  # 不断累加上半部分的价值
        # 用我们的魔法公式算出差值
            diff = abs(total_sum - 2 * current_sum)
            if diff < min_diff:
                min_diff = diff  # 遇到更小的差值，立刻更新记录！
            
    # 6. 模拟【竖着切】（切列）
        current_sum = 0  # 重新清零，准备竖着切
    # 同理，最多切到倒数第二列（m-1）
        for j in range(m - 1):
            current_sum += col_sums[j]  # 不断累加左半部分的价值
            diff = abs(total_sum - 2 * current_sum)
            if diff < min_diff:
                min_diff = diff
            
    # 7. 交卷！打印出那个最小的差值
        print(min_diff)

    if __name__ == "__main__":
        main()
    ```
