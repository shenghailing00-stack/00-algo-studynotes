# 迷宫寻路可视化器

## 项目简介

这是一个使用 Python 标准库实现的迷宫寻路可视化小项目。项目通过网格地图展示 BFS 算法从起点到终点寻找最短路径的过程，适合作为算法学习笔记仓库中的展示型项目。

## 技术栈

- Python 3
- tkinter
- collections.deque
- random

项目不使用第三方依赖。

## 功能说明

- 支持在网格中设置起点、终点和障碍物。
- 支持使用橡皮擦清除单个格子。
- 支持点击“开始寻路”运行 BFS 最短路径搜索。
- 支持点击“清空地图”恢复空白网格。
- 支持点击“随机障碍物”自动生成迷宫障碍。
- 使用颜色显示不同状态：
  - 绿色：起点
  - 红色：终点
  - 深色：障碍物
  - 蓝色：BFS 搜索过的格子
  - 黄色：最终最短路径

## 运行方式

在项目目录下运行：

```bash
python main.py
```

也可以从仓库根目录运行：

```bash
python projects/pathfinding-visualizer-python/main.py
```

## 核心算法说明

这个项目中的网格可以看作一个二维数组，每个格子保存一种状态，例如空地、起点、终点或障碍物。

从图论角度看，每个格子都可以看作图中的一个节点。上下左右相邻且不是障碍物的格子之间，可以看作存在一条边。

BFS 使用队列逐层扩展搜索范围：

1. 先把起点加入队列。
2. 每次从队列头部取出一个格子。
3. 检查它上下左右四个相邻格子。
4. 如果相邻格子没有访问过，并且不是障碍物，就加入队列。
5. BFS 第一次到达终点时，得到的就是无权图中的最短路径。

为了还原路径，程序使用 `prev` 字典记录每个格子的前驱节点。例如 `prev[next_cell] = current_cell` 表示 `next_cell` 是从 `current_cell` 走过来的。

当 BFS 找到终点后，从终点开始不断沿着 `prev` 反向回溯，直到回到起点。最后把这段路径反转，就得到了从起点到终点的最短路径。
